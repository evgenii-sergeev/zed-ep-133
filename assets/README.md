# Assets

Screenshots referenced by the root `README.md`. Drop the files here with these exact names:

| File           | Theme            | Appearance |
| -------------- | ---------------- | ---------- |
| `preview.webp` | hero / banner    | any        |
| `display.webp` | EP-133 Display   | dark       |
| `chassis.webp` | EP-133 Chassis   | light      |

## Recommended specs

- **Format:** `.webp` (smaller than PNG; what catppuccin/zed uses). PNG also works if you change the README links.
- **Width:** ~1200–1600px. Capture a real editor window, not the full desktop.
- **Content:** open a code file (so syntax highlighting shows) plus the file tree / status bar.
